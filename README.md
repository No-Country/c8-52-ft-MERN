Introducción
Midife es un hotel, ubicado en Málaga, España. Es utilizado a través de su página web para poder visualizar las habitaciones, tarifas y poder realizar reservas.

Documentación
•	Lee la guía Primeros pasos si eres nuevo en Hotel Midife.
•	Prueba la página web del Hotel Midife para comenzar a utilizarla.
•	Descubre las increíbles y confortables habitaciones de Midife en detalle.
•	Selecciona la habitación deseada y a continuación oprime el botón: Reservar.
•	Luego selecciona las fechas de entrada y salida para realizar las reservas.
•	Visualiza las tarifas y medios de pago para corroborar que todos los datos sean correctos, y después oprime el botón: Reservar ahora.
•	Confirma la reserva
